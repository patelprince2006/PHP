## SGP

title :- TripNexus

Project Description :- TripNexus is an intelligent travel management platform that integrates destination recommendation, hotel booking, train information and tracking (“Where is My Train”),and an 
                       AI-powered chatbot into a single system. The project is designed to simplify travel planning by providing users with personalized suggestions and real-time travel information.
                       The system analyzes user preferences such as budget, location, travel dates, and interests to recommend suitable travel destinations and hotels. It allows users to search and book hotels, 
                       check train schedules, and track live train status through an integrated train information module. An AI chatbot acts as a virtual travel assistant, answering user queries related to 
                       hotel availability, train timings, booking details, and destination guidance. This reduces manual searching and improves user experience by providing instant responses
